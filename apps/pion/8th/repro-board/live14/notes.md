# ref_14 재현 노트 (멤버십 안내 페이지, 816×1280)

## 섹션 분해
1. **라이프스타일 사진 풀블리드** (y 0–410): 와인·꽃병·테이블 씬. 원본 PIL 크롭(`photo_top.png`, 816×411) `<img>` 재사용.
2. **타이포 히어로** (y ~460–580): 세리프 대문자 "MEMBERSHIP"(다크, letterspaced) 위에 크림 필기체 "uptownholic"이 좌측에서 -8° 상승 각도로 겹치는 로크업. 전부 HTML 텍스트.
3. **안내 문구 2줄** (y 607–656): Pretendard, 중앙 정렬.
4. **라인아이콘 2×2 그리드** (y ~740–1204): 셀 사이 점선 십자(세로 x=406 / 가로 y=969). 아이콘 4종 전부 인라인 stroke SVG로 직접 드로잉 (쿠폰+반짝이 / 동전스택+$코인 / FREE 트럭 / 리본 선물상자). 각 셀 = 아이콘 + 볼드 제목 + 설명.

레이아웃은 816×1280 relative 컨테이너에 전 요소 절대좌표 배치(실측값 그대로).

## 실측 수치 (원본 px)
- 배경: `#e7e4dd` (231,228,221). 사진 하단 경계 y=410.
- **MEMBERSHIP**: bbox (176,503)–(640,552), cap height 49, 폭 464, 중앙(cx 408). → Playfair Display 400 self-host woff2, font-size 66.6px, letter-spacing 4.38px. 렌더 실측 (176,502,640,550) 일치.
- **uptownholic**: warm-mask bbox x 87–373, y 464–576. → Great Vibes 80px, rotate(-8deg), color `#e0c9a2` (원본 코어색 실측 217,199,163).
- **안내문구**: 폭 295px 실측 → Pretendard 18.6px (캘리브레이션 렌더로 폭 역산), line stride 32px, 글리프 top 607/639.
- **점선**: 세로 x=406 (y 746–1203), 가로 y=969 (x 203–632). 대시 4px/갭 3px, `#c1beb7`.
- **셀 중심**: 좌 cx≈275, 우 cx≈523 (좌우 비대칭 — 원본 그대로).
- **아이콘 박스**: 쿠폰 (228,739) 94×94 / 동전 (481,748) 86×85 / 트럭 (234,1001) 84×71 / 선물 (486,1004) 76×76. 스트로크 원본 ~4px 상당 → viewBox 100 기준 4.1 (선물 5.8).
- **제목**: Pretendard 700. 셀2·3·4 = 21.5px (글리프 h19, "구매 적립금" 폭 97 역산). **셀1만 17.8px** (원본이 실제로 작음: h16, 폭 94).
- **설명**: 셀2·3·4 = 18.4px / lh 21.5px ("70,000원 이상" 폭 109 역산). **셀1만 15.4px / lh 17px** ("첫구매 무료배송" 폭 96 역산, 3줄 stride 17). 색 `#4a4741`.
- 제목 top: 856/856/1110/1130, 설명 top: 888/894/1148/1169 — 하단 두 셀은 원본 자체가 베이스라인 비정렬(아이콘 높이·줄수 차이).

## 검증 절차
폰트 캘리브레이션 페이지(@100px 폭 실측)로 각 문자열 크기 역산 → 렌더 후 원본 bbox와 픽셀 대조 → 2차 보정(위치 ±2px, 셀1 폰트 축소, 아이콘 스트로크 4.6→4.1, 선물 리본 루프 타이트하게, script 색·크기). 최종 렌더 주요 bbox 오차 ≤2px.

## 재현 한계
- **필기체**: 원본 스크립트체는 Great Vibes와 유사하나 동일 서체는 아님 — 원본이 획이 약간 더 가늘고 루프가 김. 크기·기울기·색은 실측 일치.
- **세리프**: Playfair Display로 판독·재현. 원본은 헤어라인 대비가 약간 더 큼(디돈 계열 가능).
- **아이콘**: 라인아이콘 4종은 SVG 프리핸드 재작도 — 구도·비례·스트로크 굵기는 맞췄으나 세부 곡률(선물 리본 보우, 동전 코인 내부 브로큰링 각도)은 근사.
- **Playfair cap-height 이슈**: Google Fonts CDN이 헤드리스 스크린샷 타이밍에 로드 안 되는 문제 → woff2 self-host(`fonts/`)로 해결. Pretendard는 CDN 유지(프로토콜 명시).
- 하단 트림은 프로토콜 규칙(마지막 콘텐츠 행+2)에 따라 y=1206에서 컷 — 원본은 그 아래 ~74px 여백이 더 있음.
