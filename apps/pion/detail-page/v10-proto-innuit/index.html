✅ 해결됨: 없는 자료 → 할루시네이션 (밑창 예시)

피처 카드 "04 미끄럼 방지 아웃솔"에서 우리가 가진 제품 자료에 밑창샷이 없는데 텍스트로 요청하니 모델이 밑창을 지어냈습니다(할루시네이션). 이걸 막으려면 생성 전에 자료 보유 여부를 데이터로 판정하고 디렉팅하는 층이 필요:
  • D1 (자료 있음) → 실제 샷을 i2i 레퍼런스로 그대로 사용
  • D2 (자료 없음) → 절대 지어내지 않음. 디렉터가 ①다른 보유샷으로 대체 ②지원 가능한 카피로 교체 ③생략 중 결정 → 그 보정안으로만 생성
→ 위에서 검증 완료. 이제 모든 카드에 디렉터를 걸어 v8-2·풀 v10 생성으로 확장.
