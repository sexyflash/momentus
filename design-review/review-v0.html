00 왜 이렇게 짓나

전시장이 아니라, 나 대신 일하는 본체

주인공은 강형모가 아니라 방문자예요. "이거 만들었어요"가 아니라 "당신 이거 하고 싶었죠? 여기 됩니다." 도구가 밖에서 영업하고, 허브(the-moment.us)는 그들을 회수합니다.

원칙 01

즉시 체험 > 설명

버튼 하나 드래그 → "어 진짜 되네" → "이렇게 간단해?". 제품이 곧 광고. 기능 자랑 아님.

원칙 02

붙잡지 말고 갈고리

떠나도 됨. 가져간 도구 안에 귀환 버튼을 심는다. 애원 없이 효용으로 재흡입.

원칙 03

작은 무료 도구 다작

지그(한 벌 틀)에 끼워 매주 찍어낸다. 법정·FAQ·갈고리 자동 내장.

퀵팡 교훈이 증거: 작년 이 도구로 1만 팔로워가 왔는데, 귀환 갈고리가 없어서 다른 제품으로 컨버전을 못 하고 흘려보냈어요. 이번 설계의 핵심이 그 누수를 막는 것.
01 이런 느낌들이 있어요

벤치마킹 — 훔쳐올 것

전 사이트 2026-07-08 실측. 각 "느낌"과 구체적으로 뭘 가져올지. 마음에 안 드시면 여기서 방향을 다시 고르면 됩니다.

catnose.me

조용한 1인 개발자 로그

작은 프로필 카드 + 날짜 박힌 "만든 것" 타임라인. 필터 칩(전체/릴리즈/글). 자랑 없이 살아있음.

  • → 각 도구 출시 = 타임라인 한 줄
  • → 물색 블루 1개만 강조
  • → 명상적 여백 리듬
levels.io

압도적 누적 = 설득

100+ 항목을 연도별로 쌓음. "이 한 사람이 이걸 다 만들었어." 개수 자체가 무기.

  • → "지금까지 N개 무료도구" 카운터
  • → 썸네일+블루 링크 밀도
  • → 연도 헤더로 깊이감
panic.com

장난기 있는 제품 선반

카테고리 헤드 → 제품 카드 스택. 카드 계약 통일(제목→한줄→플랫폼→CTA). 개성은 카피에.

  • → her·ChatPage를 "제품"처럼
  • → 위트는 문장, 프레임은 미니멀
  • → 4파트 카드 = 재사용 부품
goodenough.us

정직한 목소리 (안티 마케팅)

좁은 글 칼럼, 히어로 이미지 없음. 시처럼 끊기는 줄바꿈. * * * 구분자. 대화체가 곧 정체성.

  • → 소개·빈 상태 카피 톤
  • → "AI 미니멀"이 차가워지지 않게
  • → 이게 모멘터스가 말하는 법
10015.io

랜딩이 곧 도구 런처

강조 키워드 히어로 + 아이콘 카드 그리드. 카테고리 아이콘 색만 색 에너지. "와 되네"가 1클릭 거리.

  • → 도착→1클릭 체험
  • → 아이콘 색만 중립필드에
  • → 카테고리별 "전체보기 →"

추천 ★A안

THE CHANGELOG

catnose 척추 + levels 밀도 + goodenough 목소리. 작은 창업자 카드 + 살아있는 "만든 것" 타임라인. 도구마다 날짜 한 줄.

위험: 너무 조용 → 미완성 블로그처럼. → 히어로에 1클릭 작동 도구를 접목해 해소.

B안

THE ONE BOX

10015 히어로 + panic 카드. 랜딩이 곧 런처. 카테고리 아이콘 색이 유일한 색 에너지. 체험이 스크롤 위.

위험: 일반 도구 디렉토리처럼 보여 1인 창업자 영혼 상실. → 얼굴+정직한 목소리 상주로 해소.

C안

EXPERIMENTAL STUDIO

큰 타입 주도 + 블루→잉크 그라디언트 1회 + 인라인 마이크로 데모. mono 악센트로 "AI 실험적" 엣지.

위험: 큰타입+그라디언트=AI 흔한 느낌으로 미끄러짐. → 그라디언트 1개·모션 1개로 배급.

권고

A안(체인지로그)을 척추로, B안 히어로(1클릭 작동 도구)를 접목. 이러면 형 퍼널 — 도착 → 무료도구 체험 → "와 되네" → 누적 타임라인 → 팬 — 이 정확히 굴러가고, goodenough 목소리 + 창업자 증거로 "디렉토리"가 아니라 명백히 강형모의 스튜디오가 됩니다. 아래 홈·상세가 이 조합이에요.
