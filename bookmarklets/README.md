# MOMENTUS 북마크릿 도구 (관리 규약)

> 북마크릿 = 설치 없이 "버튼 하나를 크롬 북마크바에 드래그"하면 끝나는 초경량 도구.
> 소스는 여기(momentus repo)에서 관리하고, 랜딩/배달 페이지는 `apps/<도구>/`에 둔다.
> 근거: 북마크릿은 가볍고 랜딩과 1:1로 짝지어지므로 무거운 독립앱(→ Seagate)과 달리 사이트 repo 안에서 관리하는 게 일관적. (전략: `docs/APEX_STRATEGY.md` §7 지그)

## 라인업 (4종, 전부 무료)

| 도구 | 파일 | 하는 일 | 상태 |
|---|---|---|---|
| 인스타 인기순 정렬 | `insta-rank.js` | 인스타 게시물을 좋아요순으로 재정렬 (레퍼런스 리서치) | 준비됨 |
| 유튜브 인기순 정렬 | `youtube-rank.js` | 검색·채널·쇼츠를 조회수순 정렬 | 준비됨 |
| 핀터레스트 원본 추출 | `pinterest-grab.js` | 저해상 썸네일 대신 originals 원본 URL 골라 일괄 복사 | 준비됨 |
| 퀵팡 (쿠팡 퀵보기) | `apps/quickpang/` (기존) | 상품 클릭 없이 옵션·품절 여부 즉시 확인 | 라이브 검증(1만 팔로워) |

## 소스 관리 규칙

1. **원본 소스 = 여기 `.js` 파일** (읽기 좋은/편집용). `javascript:` prefix 없이 순수 함수 본문.
2. **배포 산출물 = `javascript:` URL** — 빌드(minify + URL-encode)해서 `apps/<도구>/index.html`의 드래그 버튼 `href`에 주입.
3. **공용 스킨** — 4개 랜딩은 같은 컴포넌트(드래그 버튼 + FAQ + 귀환 갈고리 + 연관 도구 링크)를 공유. 스킨만 갈아끼움. (지그)
4. **귀환 갈고리 필수** — 각 도구 결과 UI 구석에 "⚡ 다른 무료 도구 보기 → the-moment.us" 삽입. QuickPang 교훈(1만 트래픽을 컨버전 못 함)의 해법.
