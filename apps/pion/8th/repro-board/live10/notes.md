좌측 열 선택. 브랜드 헤드(세리프+고딕 믹스), 캡션 2단, 대형 넘버+저스티파이 본문, 사선 장치,
서브헤드, 오프셋 이미지, Gallery 라벨+헤어라인 룰까지 — 3열 중 디자인 장치가 가장 온전하게
한 열에 다 들어있고, 상하가 잘리지 않아 실측이 가장 정확함 (우측 열은 하단이 이미지 경계에서 크게 잘림).

## 구조·디자인 장치 분해
1. 세리프(WINDSOR)와 고딕(THE/CHAIR)을 겹쳐 쌓은 브랜드 로크업, 중앙 정렬
2. 풀블리드에 가까운 사진 (여백 5px < 텍스트 여백 6px)
3. 마이크로 캡션 2단 그리드 — 라벨(좌) / 값(우), 값 칼럼 x54가 본문 칼럼과 공유됨
4. 대형 숫자 + 좁은 저스티파이 칼럼의 비대칭 2단 (x54 그리드 라인이 열 전체를 관통)
5. ↗ 사선 = 섹션 전환 장치 (넘버 섹션 → 서브헤드)
6. 오프셋(우측 치우침) 소형 이미지로 리듬 변화, 역시 x54 라인 정렬
7. Gallery 라벨 + 헤어라인 룰 → 풀폭 갤러리 사진

## 재현 방법
- 134px 고정 스트립, 전 요소 absolute 배치 (실측 y/x 좌표 직입력)
- 사진 3장 전부 원본 PIL crop → `photo1.png`(123×186), `photo2.png`(64×96), `photo3.png`(124×115), `<img>` 재사용. i2i 없음
- 사진 안에 박힌 텍스트 없음 → 배경 대체 불필요
- 폰트: 고딕 = Pretendard CDN, 세리프 = **Playfair Display 600** (고대비 디도네 계열로 원본과 가장 유사)
- 본문 13행을 원본과 동일한 위치에서 `<span>` 분리 + `text-align-last:justify; text-justify:inter-character`
  → 원본의 글자단위 벌림 저스티파이 재현. 마지막 행("mind.")만 좌측 정렬
- 보정 1회: WINDSOR 자간 2.5→0.7px (실측 w106→93 목표), 본문 8→7.5px + 강제 개행(자연 개행이 15행으로 늘어졌음),
  캡션 6→5.5px, Gallery 11→10px, 미세 y 오프셋 (±2px)
- 렌더 후 검증: 본문 블록 마지막 행 렌더 y452–457/x55–70 vs 원본 y453–457/x54–69
