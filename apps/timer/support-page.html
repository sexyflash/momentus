Purchasing & Subscription

What's included in Timer Pro?
Timer Pro includes advanced features such as:
  • Unlimited timer settings
  • Advanced notification features
  • Perfect UI customization
  • Menubar timer functionality
  • Custom alert messages and sounds
You can try Timer Pro free for 14 days with our trial period.
Can I cancel my subscription anytime?
Yes, you can cancel your Timer Pro subscription at any time through your App Store account settings. After cancellation, you'll continue to have access to Pro features until the end of your current billing period.
How do I restore my purchase?
To restore your purchase, simply click the "Restore Purchases" button in the upgrade screen. Make sure you're signed in with the same Apple ID you used to make the original purchase.

Features & Usage

How do I customize the timer appearance?
With Timer Pro, you can customize:
  • Main text color
  • Sub text color
  • Button color
  • Progress bar color
  • Background color and opacity
  • Font style
Access these options in the Settings window under the Timer tab.
How do notifications work?
Timer Pro offers several notification options:
  • Start notifications
  • Halfway point notifications
  • 10-second countdown notifications
  • Completion notifications
  • Custom alert messages and sounds
You can configure these in the Settings window under the Alerts tab.
Can I use the timer in the menu bar?
Yes, Timer Pro includes a menu bar timer feature. You can enable or disable this in the General Settings tab. The menu bar timer shows the remaining time and updates in real-time.

Technical Support

How do I keep the timer always on top?
The "Always on Top" feature is available for Pro users. You can enable this option in the General Settings tab. When enabled, the timer window will stay above other windows on your screen.
How do I change the timer size?
You can choose from three different size options in the Timer Settings:
  • Small - Compact view
  • Default - Standard size
  • Large - Enhanced visibility
The size setting affects all elements of the timer proportionally.
